# Food-Ordering

This is a food delivery system

## Getting Started

Install Dependencies
```
    npm install -r requirements.txt
```

Activate Enviroment
``` 
    source path/to/env/bin/activate
```

Run Local Server 
```
    python manage.py runserver <port{9001}>
```

Navigate to GraphQL endpoint
```
    127.0.0.1:9001/graphql
```
